# LLM Panel Usage Guide

## Quick Start

### Activating the Panel
1. Open any note in the Notes editor
2. Look at the top-right header next to "History" button
3. Click the **"AI Insights"** button (💡 icon)
4. The Related Discussions panel slides in from the right

### Using the Panel
- **Start Writing**: Begin typing or pasting content into your note
- **Automatic Search**: Once you have 20+ characters, the panel automatically searches for similar threads
- **View Suggestions**: Related discussions appear with similarity scores
- **Color Coding**:
  - 🟢 Green (80%+): Highly relevant
  - 🔵 Blue (70-79%): Very relevant  
  - 🟠 Amber (60-69%): Somewhat relevant
  - ⚪ Gray (below 60%): Lower relevance

### Viewing Related Threads
- Each card shows:
  - **Title**: The thread's main topic
  - **Description**: Excerpt from the thread
  - **Reply Count**: How many responses
  - **Vote Score**: Community engagement score
  - **Panel Type**: 📚 Academic or 💼 Alumni

### Navigating to a Thread
- Click on any thread card to open it in full detail view
- The panel will remain open - you can navigate back to your note
- Discussion insights appear in a new page

### Closing the Panel
- Click the **X button** in the panel header
- Click the "AI Insights" button again
- Opening History or Share panels automatically closes it

## Features

✨ **Real-time Search**: Results update as you type (with 350ms debounce)
🎯 **Smart Threshold**: Only searches with meaningful content (20+ chars)
⚡ **No Chatbot**: Pure semantic similarity linking, not conversational
🔄 **Live Updates**: Works across all note collaborators simultaneously
📱 **Responsive**: Adapts to different screen sizes
🎨 **Polished UI**: Smooth animations and hover effects

## Tips

1. **Better Results**: Write more detailed notes for more accurate suggestions
2. **Multiple Panels**: You can compare with History and Share panels too
3. **Drafts**: Works with drafts and unsaved content
4. **Collaboration**: Related threads show regardless of who's editing

## Troubleshooting

| Issue | Solution |
|-------|----------|
| No suggestions appearing | Write more content (minimum 20 characters) |
| Suggestions seem irrelevant | The search uses semantic meaning; try rephrasing |
| Panel is slow to respond | Check your internet connection; backend may be processing |
| Can't click a thread | Make sure you're clicking within the thread card, not the edge |
